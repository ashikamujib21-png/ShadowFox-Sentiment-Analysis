# 🐦 X (Twitter) Sentiment Analysis — ShadowFox Internship

## 📌 Project Overview
This project performs **Sentiment Analysis on X (Twitter) data** using
Natural Language Processing (NLP) techniques. Tweets are classified as
**Positive, Negative, or Neutral** using the VADER sentiment engine,
followed by statistical analysis and rich visualizations.

## 🎯 Objective
To gain insights into prevailing public sentiments expressed on X by
analyzing tweet content across multiple categories.

## 🛠️ Tech Stack
- **Language** : Python
- **Platform** : Google Colab
- **Libraries** : Pandas, NumPy, NLTK, VADER, Matplotlib, Seaborn, WordCloud, FPDF
